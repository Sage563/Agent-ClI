Community Guidelines

We welcome contributions of all kinds, from bug reports and documentation updates to new features and architectural improvements.

How to Contribute

  1. Report Bugs: If you find an issue, please open a GitHub Issue with clear steps to reproduce.
  2. Suggest Features: Have an idea? Start a discussion or open an issue labeled as a feature request.
  3. Pull Requests: Fork the repo, create a branch, and submit a PR. Please ensure all tests pass and code is linted.
  4. Documentation: Help us improve the website and docs by submitting fixes to the docs/ folder.
