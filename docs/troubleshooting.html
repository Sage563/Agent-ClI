Troubleshooting

Common issues encountered by users and their recommended solutions.

Streaming Timeouts

If the AI response hangs or cuts off prematurely, try increasing the stream timeout:

/config stream_timeout_ms 30000

Authentication Failures

Ensure your API keys are correct and you have active billing on your provider account. You can verify your active keys with:

/config openai_api_key

Permission Denied

If Agent CLI is unable to read or write files, ensure it has the correct permissions for the current workspace. In Mission Mode, it may require a higher session access grant:

/access full

Log Files

If you encounter a persistent crash, check the agent logs in your system's AppData directory. You can also run the agent with the --verbose flag to see real-time debug output in your terminal. 

agent-cli --verbose

System-Specific Issues (Windows)

On Windows, ensure you are running in a terminal that supports ANSI colors (like Windows Terminal or PowerShell 7) for the best TUI experience. If symbols are not displaying correctly, install a "Nerd Font" like JetBrains Mono NF.
