AI Providers & Models

Agent CLI is provider-agnostic. You can easily switch between local open-source models and premium remote APIs.

Supported Providers

Local (Ollama)

Run models like Qwen 2.5/3, Llama 3, or DeepSeek locally.

OpenAI

High-performance models like GPT-4o and o1-mini.

Google Gemini

Gemini 1.5 Pro and Flash with large context windows.

Anthropic

Claude 3.5 Sonnet, known for exceptional coding ability.

DeepSeek

Powerful and cost-efficient coding models.

How to Switch

Use the /provider command followed by the provider ID:

/provider anthropic
/model claude-3-5-sonnet-20241022

Advanced Model Parameters

You can fine-tune how the models behave using core configuration keys. These are passed to the provider during every generation turn.

  • provider_temperature: Controls randomness (0.0 to 1.0).
  • provider_top_p: Nucleus sampling parameter.
  • provider_max_tokens: Hard limit on AI response length.
/config provider_temperature 0.2
/config provider_max_tokens 8000
