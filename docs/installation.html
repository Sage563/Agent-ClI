Installation

Get Agent CLI up and running on your local machine in minutes.

Prerequisites

  • Node.js: Version 20.0.0 or higher is required.
  • npm: Node package manager for installing dependencies.
  • Git: Required for cloning the repository.

Step 1: Clone the Repository

git clone https://github.com/Sage563/Agent-ClI.git
cd Agent-ClI

Step 2: Install Dependencies

Run the following command in the project root to install all required libraries:

npm install

Step 3: Verify the Installation

Start the agent in development mode to ensure everything is working correctly:

npm run dev

Common Installation Issues

EACCES Error: If you see permission errors during npm install, avoid using sudo. Instead, use a version manager like nvm to manage your Node.js installation.

Python Dependency: Some underlying libraries (like node-gyp) may require Python to be present on your system for compilation of native modules.

Post-Installation Next Steps

Once installed, you should proceed to the Setup Guide to configure your AI providers and environment variables.
