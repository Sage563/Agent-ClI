Configuration Reference

Every aspect of Agent CLI can be fine-tuned. This page is a complete reference of every available configuration option.

General Settings

active_provider string

The ID of the currently active model provider. (Default: "ollama")

run_policy "ask" | "always" | "never"

Determines if the agent requires permission before executing shell commands. (Default: "ask")

stream boolean

Enables or disables standard text streaming for AI responses. (Default: true)

max_budget number

The lifetime dollar budget for API usage in the current session. (Default: 0 - unlimited)

User Interface Settings

ui_theme "dark" | "light" | "terminal"

Changes the color scheme of the TUI. (Default: "dark")

show_thinking_logs boolean

Displays the internal steps the agent is taking during planning. (Default: true)

audio_feedback boolean

Enables text-to-speech feedback for major agent actions. (Default: false)

Provider-Specific Settings

Ollama

ollama_endpoint string (URL)

The URL of your local Ollama instance. (Default: "http://localhost:11434")

ollama_model string

The specific model tag to use for generation. (Default: "qwen3:14b")

OpenAI

openai_api_key string (API Key)

Your OpenAI secret API key.

openai_model string

The model ID to use (e.g., "gpt-4o").

Security & Access

allowed_paths string[]

An explicit allow-list of directories the agent is allowed to access. (Default: [current workspace])

env_bridge_enabled boolean

If true, automatically loads API keys from your local .env file. (Default: true)

Environment Variables

You can also set many of these via environment variables:

  • AGENT_PROVIDER: Overrides active_provider.
  • AGENT_POLICY: Overrides run_policy.
  • AGENT_DEBUG: Enables verbose logging if set to true.
